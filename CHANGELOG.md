# Changelog

## 3.0.3 - 2026-06-18

- Keeps the actively dialed contact pinned at the top of the call list until the call ends.
- Removes the active contact from the call list only after call-end detection, so reps can always see who is currently being called.
- Adds a visible countdown for the next auto-dial when auto-dial-on-end is enabled.
- Improves active-call display for long contact names with hover/focus scrolling and full-name tooltips.
- Pins the extension manifest key to the Chrome Web Store extension ID for local and release builds.
